✔ Github
- Git Desktop 대신 Gitbash를 쓰기 시작했다. 이유는 깃데스크탑은 켜지는 데 시간이 걸려서 답답하다.
- 커밋하는법
1. 로컬 레파지토리로 이동
2. `git add .` : 작업공간으로 해당 폴더의 모든 파일(중 바뀐 파일들만?)을 이동
3. `git commit -m "메세지"` 로 커밋한다.
4. `git push -u origin master` 로 푸시한다.
